# Witness
커플 디데이 및 버킷리스트 작성앱 입니다.<br/>
플레이 스토어에 등재되어 있으며, 디데이 계산 + 기념일 추가 + 기념일 남은 일자 계산 등의 기능들이 있습니다.<br/>
또한 버킷리스트 기능이 있어 서로 하고싶은 내용들을 적어 관리 할 수 있습니다.

<br>

### 프로젝트 개발기간
***
2020-09-01 ~ 2020-10-04  

<br>

## 기술 스택

FE : Android Java

DB : Sqlite
